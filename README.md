# Claulancer - Professional Freelance Marketplace

A modern freelance marketplace platform connecting talented freelancers with clients worldwide.

## Features

- **Authentication System**: Role-based authentication for freelancers and clients
- **Dual Profile Views**: Separate profile interfaces for freelancers and professional clients
- **AI-Powered Recommendations**: Smart job matching and talent recommendations
- **Modern UI/UX**: Beautiful animations and responsive design
- **Skills Marketplace**: Browse opportunities across CSE & IT domains

## Technology Stack

- HTML5, CSS3, JavaScript
- Deployed on Vercel
- GitHub for version control
